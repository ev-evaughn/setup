# Setup

## Software
These will be installed or updated to the following versions
```
bash-5.2
cmake-3.31.2
gcc-14.2
gdb-15.2
git-2.47.1
make-4.4.1
nvim-0.11.0
```

## Config Files
These will be overwritten
```
~/.bash_profile
gdbinit
~/.gitconfig
init.lua
```

## Prerequisites
```cmake``` already installed and in ```$PATH```

## To run
In setup's root directory
```
cmake --build
```
